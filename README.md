# Social Connect Hub

A unified platform to discover and connect with influencers across Instagram, YouTube, and Facebook.

## Features
- User login & authentication
- Influencer profiles with follower counts and promotions
- Search & filter influencers
- Favorites management
- Admin dashboard (extendable)

## Tech Stack
- Flask (Python backend)
- SQLite (database)
- HTML, CSS, JS (frontend)
- Gunicorn (production server)

## Deployment
- Push to GitHub
- Deploy on [Render](https://render.com) or [Vercel](https://vercel.com)
- Render Start Command: `gunicorn app:app`
